# Sistema de Gestión de Inventario - Django

## Enlace de Producción
**https://inventory-ke5c.onrender.com**

## Descripción General
Sistema web completo para la gestión integral de inventarios, desarrollado con Django. Controla todas las operaciones de un negocio: compras, ventas, inventario, catálogos y análisis en tiempo real.

## Características Principales

### Dashboard Central
- Resumen ejecutivo con KPIs clave
- Gráficos interactivos de ventas
- Alertas automáticas de stock bajo
- Actividad reciente del sistema

### Gestión de Inventario
- Control completo de existencias
- Movimientos de entrada/salida
- Ajustes de inventario
- Categorización de productos

### Módulo de Ventas
- Proceso completo de venta
- Facturación electrónica
- Gestión de clientes
- Historial de transacciones

### Módulo de Compras
- Órdenes de compra a proveedores
- Seguimiento de pedidos
- Gestión de proveedores
- Control de costos

## Tecnologías Utilizadas

### Backend
- **Framework:** Django 
- **Base de datos:** PostgreSQL

- **Autenticación:** Django Allauth

### Frontend
- **HTML5, CSS3, JavaScript**
- **Bootstrap 5** - Diseño responsivo
- **Chart.js** - Gráficos interactivos
- **DataTables** - Tablas avanzadas

### Despliegue
- **Plataforma:** Render.com
- **Servidor:** Gunicorn
